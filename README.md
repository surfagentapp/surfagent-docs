# SurfAgent Docs

Official documentation for [SurfAgent](https://surfagent.app), the Windows AI browser agent that gives your AI a real local Chrome instance with persistent sessions.

## Start here

If this repo feels confusing, use this flow:

1. [Start Here](./docs/start-here.md)
2. [Getting Started](./docs/getting-started.md)
3. [Connect Your AI Agent (MCP Server)](./docs/mcp-server.md)
4. [Skills, Adapters, and MCP: What to Use When](./docs/skills-and-adapters.md)
5. [FAQ & Troubleshooting](./docs/faq.md)

That is the shortest path from zero to a working setup.

## What this repo is for

`surfagent-docs` is the canonical public docs repo for:
- install and setup docs
- product explanations
- licensing and update docs
- troubleshooting and FAQ
- public repo ownership and navigation
