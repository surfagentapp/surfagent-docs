# Start Here

If you're new to SurfAgent, use this page as your map.

## What SurfAgent actually is

SurfAgent is a Windows desktop app that gives your AI agent access to a real local Chrome browser with persistent sessions.

That means:
- your AI can browse with a real Chrome profile
- cookies and logins persist
- browser traffic stays on your machine
- you can connect Claude Code, Codex, Cursor, Windsurf, Hermes, OpenClaw, or any MCP client

## One important distinction

Most users only need:
- the SurfAgent app
- the `surfagent-mcp` package
- maybe a skill or two

Most users do **not** need every public repo on day one.

That detail mainly matters for contributors, advanced users, and people wiring custom flows.
